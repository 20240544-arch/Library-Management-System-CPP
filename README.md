Library Management System (OOP)
This is a C++ project that demonstrates core Object-Oriented Programming concepts including:

Inheritance & Polymorphism (Book, Magazine, ResearchPaper).

Encapsulation (Private, Protected, Public members).

Friend Functions for price comparison.

Static Members to track the total number of items.
